## Featured Projects

### ARKU — Sales Monitoring System
A sales operations platform with a cross-platform mobile app for Sales Representatives and a web-based admin panel. SRs can register shops, create and manage orders, and handle deliveries on the go. Admins get a centralized dashboard to monitor orders, track employee attendance, manage staff, configure products, create promotional offers, and publish news.

### PickNPack — Inventory Management System
A full-cycle inventory system built for small and medium businesses. Supports role-based access control where a superadmin can create users and assign granular module permissions per role. Core workflows include Purchase Orders (PO), Sales Orders (SO), invoicing, product tracking, and document printing.
